# Customer Churn Prediction System

This project predicts whether a bank customer will churn using a Neural Network.

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit

## How to Run
1. Create and activate virtual environment
2. Install requirements
3. Train model:
   python src/train_model.py
4. Run application:
   streamlit run app.py
